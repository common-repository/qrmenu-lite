== Frequently Asked Questions ==

= The menu page returns 404 error. How to solve it? =

Go to Settings - Permalinks and hit the Save button, to reset the permalinks. This will solve the problem.

= How do I install QRMenu Lite? =

There are two ways you can install QRMenu Lite plugin.

You can install the plugin from your WordPress dashboard:
1. Navigate to your WordPress Dashboard -> Plugins -> Add New
2. Search for QRMenu Lite plugin
3. Install and Activate QRMenu Lite from your Plugins page.

Or you can install it directly from WordPress.org
1. Locate and download QRMenu Lite on wordpress.org
2. Upload the qrmenu-lite directory to your ‘/wp-content/plugins/’ directory using your preferred method (ftp, sftp, scp, etc…)
3. Install and Activate QRMenu Lite plugin from your Plugins page.

= How do I get support? =

If you require assistance while using the QRMenu Lite plugin, you can visit the <a href="https://wordpress.org/support/plugin/qrmenu-lite/" target="_blank">support forum</a> on WordPress.org, or you can <a href="https://modeltheme.ticksy.com" target="_blank">submit a ticket</a> and our support agents will respond to you as soon as possible.

= Do I need to know how to code? =

Absolutely not. QRMenu Lite was designed to allow you to create Restaurant Menus directly from the visual dashboard of your WordPress, without the need to understand or use any coding.
